  选择题目：**短跑计时器**

  

  短跑计时器设计与实现（难度系数：0.9）

  短跑计时器描述如下：

  ◼ 短跑计时器显示分、秒、毫秒； 

  ◼ “毫秒”用两位数码管显示：百位、十位；

  ◼ “秒”用两位数码管显示：十位、个位；

  ◼ “分”用一位 LED 灯显示，LED 灯“亮”为 1 分； 

  ◼ 最大计时为 1 分 59 秒 99，超限值时应可视或可闻报警； 

  ◼ 三个按键开关：计时开始/继续（A）、计时停止/暂停（B）、复位/清零（C）
